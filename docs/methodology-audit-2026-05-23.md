# model-view Scientific and Methodology Audit

Date: 2026-05-23

Scope: README.md, plan.md, map parameter methodology tooltips, rendered hover/legend values, SoundingDrawer methodology text, sounding operational tables, renderer-derived grids, and tests after the low-cost methodology corrections in this worktree.

Audit status vocabulary:

- OK: method, unit conversion, UI label, and plan wording are aligned.
- OK with disclosed limitation: scientifically defensible and now labeled as an approximation, proxy, sparse-profile product, or direct model diagnostic.
- Corrected: an issue from the prior audit was fixed in code or wording before this audit was refreshed.
